    View Source F1Bot.F1Session.Server (f1_bot v0.6.0)

    GenServer that holds the live F1Bot.F1Session instance, acts as an entrypoint for incoming live timing packets and executes all side effects by passing event messages to F1Bot.Output.Discord and F1Bot.Output.Twitter via PubSub.
